# JogoDaMemoriaDosSantos
Este projeto é um jogo da memória desenvolvido em HTML, CSS e JavaScript, com temática voltada para santos católicos. O objetivo é treinar a memória e, ao mesmo tempo, conhecer figuras importantes da tradição cristã.

Funcionalidades:
- Cartas embaralhadas a cada partida, garantindo variedade.
- Mecânica simples de virar cartas e verificar pares.
- Imagens representando santos católicos como São José, Santa Cecília, Santo Agostinho, entre outros.
- Feedback visual imediato ao acertar ou errar um par.
- Layout responsivo e estilizado com CSS.
  
Como jogar:
- Clique em uma carta para revelá-la.
- Em seguida, clique em outra carta.
- Se as imagens forem iguais, você encontrou um par!
- Caso contrário, as cartas voltam para o verso e você pode tentar novamente.
- O jogo termina quando todos os pares forem encontrados.
  
Estrutura do projeto:
- index.html → Estrutura principal do jogo.
- css/style.css → Estilos visuais e layout.
- img/ → Pasta com as imagens dos santos e o verso das cartas.
